# Ecomly - Full Stack E-Commerce Platform

A complete e-commerce application built with Flutter for mobile and Node.js for backend.

##  Tech Stack

**Frontend (Mobile):**
- Flutter
- Dart

**Backend:**
- Node.js
- Express.js
- MongoDB

## Project Structure
```
ecomly/
├── prakhya_ecomly/              # Flutter mobile app
└── prakhya_ecomly_server/       # Node.js backend API
```

##  Setup Instructions

### Backend Setup
```bash
cd prakhya_ecomly_server
npm install
npm start
```

### Flutter App Setup
```bash
cd prakhya_ecomly
flutter pub get
flutter run
```
